About Us

BitsandGears is a resource for learning robotics (Intended for VEX students specfically, but concepts apply)
The ultimate guide and resource list for learning competitive robotics, mechanical engineering, and programming concepts.
When getting started with robotics I found it hard to find resources to build up my knowledge of mechanical engineering, programming, and in general competitive robotics. To address this problem I'm going to be building up a knowledge library, for robotics, and getting started with competitve robotics as a whole.
The biggest problem that I face is learning concepts and gaining knowledge that I am able to apply to the VEX challenge

I would like to build a community, knowledgebase, and starting point for members of the competitve robotics community.
Please contribute to the site by adding a pull request to improve documentation, or to add a new article to the site.
You may do this through the github repo

Who is this project for?

This project is for those starting out with robotics who don't know how to start, or those who would like to build their mechanical know-how to imprvove their competitve robotics game.

This project is generally meant for the competitve robotics community (Especially the VEX IQ and VRC Divisions), I hope that everyone can learn something, and that we can all contribute to a project that betters everyone's robotics smarts
